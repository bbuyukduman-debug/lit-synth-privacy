LIT-SYNTH – Gizlilik Politikası

Son güncelleme: Nisan 2026

LIT-SYNTH, kullanıcı verilerini toplamaz, satmaz veya üçüncü taraflarla paylaşmaz.

1. Toplanan Veriler

LIT-SYNTH yalnızca aşağıdaki verileri kullanır:

  • Gemini API Anahtarı: Kullanıcı tarafından girilen anahtar, yalnızca tarayıcının yerel depolama alanında (chrome.storage.sync) saklanır. Bu alan Chrome/Opera tarafından şifrelenir ve yalnızca kullanıcının kendi hesabında senkronize edilir.
  • Makale Metinleri ve Özetler: Analiz edilen makalelerin başlıkları ve özetleri, tarayıcının yerel depolama alanında (chrome.storage.local) saklanır. Bu veriler cihazı terk etmez.
  • Aktif Sekme URL'si: Uzantı, analiz işlemi başlatıldığında yalnızca kullanıcının o an açık olan sekmesinin URL'sini okur. Tarama geçmişi takip edilmez.

2. Verilerin Kullanımı

Toplanan veriler yalnızca şu amaç için kullanılır:

  • Akademik makalelerin metnini Google Gemini API'ye göndererek özet oluşturmak.

Bu süreçte makale metni, kullanıcının kendi API anahtarı aracılığıyla Google'ın sunucularına iletilir. Bu iletişim doğrudan tarayıcı ile Google arasında gerçekleşir; LIT-SYNTH'in kendi sunucusu bulunmamaktadır.

3. Üçüncü Taraf Hizmetler

LIT-SYNTH yalnızca aşağıdaki üçüncü taraf hizmetiyle iletişim kurar:

  • Google Gemini API – Makale metni analizi için. Gönderilen veriler Google'ın Gizlilik Politikası ve Kullanım Koşulları kapsamındadır.

Bunun dışında hiçbir analitik araç, reklam ağı veya harici sunucuyla veri paylaşılmaz.

4. Veri Saklama ve Silme

Kullanıcı, uzantı arayüzündeki "Belleği Sıfırla" butonu ile analiz edilen tüm makale verilerini istediği zaman kalıcı olarak silebilir. API anahtarı ise tarayıcı ayarlarından uzantı kaldırılarak silinebilir.

5. Çocukların Gizliliği

LIT-SYNTH, 13 yaşın altındaki bireylerden bilerek veri toplamaz. Bu uzantı akademik araştırma amacıyla tasarlanmış olup yetişkin kullanıcılara yöneliktir.

LIT-SYNTH – Privacy Policy

Last updated: April 2026

LIT-SYNTH does not collect, sell, or share user data with any third parties.

1. Data Collected

LIT-SYNTH only uses the following data:

  • Gemini API Key: The key entered by the user is stored exclusively in the browser's local storage (chrome.storage.sync). This storage is encrypted by Chrome/Opera and synced only within the user's own account.
  • Article Texts and Summaries: Titles and summaries of analyzed articles are stored in the browser's local storage (chrome.storage.local). This data never leaves the device.
  • Active Tab URL: The extension reads the URL of the user's currently open tab only when an analysis is initiated. Browsing history is never tracked.

2. Use of Data

Collected data is used solely for the following purpose:

  • Sending academic article text to the Google Gemini API to generate summaries.

In this process, article text is transmitted to Google's servers via the user's own API key. This communication occurs directly between the browser and Google. LIT-SYNTH does not operate its own servers.

3. Third-Party Services

LIT-SYNTH communicates only with the following third-party service:

  • Google Gemini API – For article text analysis. Data sent is subject to Google's Privacy Policy and Terms of Service.

No analytics tools, advertising networks, or external servers receive any user data.

4. Data Retention and Deletion

Users can permanently delete all analyzed article data at any time using the "Reset Memory" button in the extension interface. The API key can be removed by uninstalling the extension via browser settings.

5. Children's Privacy

LIT-SYNTH does not knowingly collect data from individuals under the age of 13. This extension is designed for academic research and intended for adult users.
